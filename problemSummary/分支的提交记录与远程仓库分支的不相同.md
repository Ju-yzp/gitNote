[拒绝切换至另外一个分支](../image/enableSwitchBranch.png)
### 原因

在这之前，我的所在分支更改了和新创建了一些文件，但是另外的一个分支并没有跟踪到这些文件，所以需要先建立跟踪先

```
git add <file>(自行替换为自己要进行跟踪的文件)
```
